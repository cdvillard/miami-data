Miami Data

An open repository for all things data in Miami-Dade County.
Created for, and curated with the help of, Code for Miami.

Open Data Portals

  • Miami-Dade County Open Data Portal

    Varying data sets like 311 (service requests), employee salaries, building permitting activity and GIS (Geographic Information Systems)

  • City of Miami Open Data Portal

    Various data sets including budget, community development and capital improvements.

  • US City Open Data Census: Miami-Dade County

    Survey of Miami-Dade County's open data sources by local residents are part of the US City Open Data Census project.

Education

  • Miami-Dade County Public Schools Assessment, Research, and Data Analysis

    Membership data in Excel format of Miami-Dade County Schools.

Transit

  • Miami Beach DecoBike docking station locations

    All DecoBike locations on Miami Beach, as an XML file that was to be read by the Google Maps Javascript API. Hidden behind a web app. Data taken from the public url: http://www.decobike.com/playmoves.xml (URL still valid as of March 2014)

  • Downtown Miami CitiBike docking station locations

    All CitiBike locations in Downtown, as XML data.

Data sets created by Code for Miami

  • 2013 Miami-Dade County Local Business Tax Data

    Repository hosting raw data for the Miami-Dade County Local Business Tax Receipts for 2013. Accessbile via API at https://github.com/Code-for-Miami/mdt-lbt-api.

Major Civic Data Sources

  • National Day of Civic Hacking Data Sets

    A central hub of datasets made available to National Day of Civic Hacking, covering everything from the county to the federal level.
